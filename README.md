# PreviewExplorer

AviUtl2 プラグイン — プレビューとメディアエクスプローラーを一体化したウィンドウ

## 機能

- **2 層タブ式エクスプローラー**: 大カテゴリ × 小カテゴリでフォルダをブックマーク管理
- **タブごとの位置記憶**: 各タブが最後に表示していた場所を個別に保持・復元
- **ナビゲーションバー**: ←→↑ ボタン、アドレスバー直接入力（Enter でナビゲート）
- **ファイル操作**: コピー・切り取り・貼り付けボタン（`IFileOperation` 使用）
- **エイリアス保存**: タイムライン選択オブジェクトを `.object` ファイルとして保存
- **ダークモード**: AviUtl2 のテーマカラーに追従
- **タブ並び替え**: 大タブ・小タブをドラッグ＆ドロップで並び替え

## インストール

1. ビルド済みの `PreviewExplorer.aux2` を AviUtl2 の `Plugin` フォルダに配置
2. AviUtl2 を起動 → プラグインウィンドウに「Preview edit + Explorer」が追加される

## ビルド

```powershell
cmake -B build -G "Visual Studio 17 2022" -A x64
cmake --build build --config Release
# build/Release/PreviewExplorer.aux2 が生成される
```

**要件**: Visual Studio 2022、CMake 3.10 以上、Windows SDK

## 使い方

### タブ操作

| 操作 | 内容 |
|---|---|
| 大タブ左クリック | ワークスペース切り替え（前回の場所を復元） |
| 大タブ右クリック | 追加 / 名前変更 / 初期位置に移動 / 削除 |
| 小タブ左クリック | フォルダブックマーク切り替え |
| 小タブ右クリック | 追加 / 名前変更 / パス変更 / 現在位置を保存 / 初期位置に移動 / 削除 |
| タブドラッグ | 順序変更 |

### ナビゲーション

| 操作 | 内容 |
|---|---|
| `←` / `→` | 履歴を戻る / 進む |
| `↑` | 親フォルダへ |
| アドレスバー + Enter | パスを直接入力して移動 |
| Ctrl+C / X / V | ファイルのコピー / 切り取り / 貼り付け |

### エイリアス保存（A保存）

1. AviUtl2 のタイムラインでオブジェクトを選択
2. **A保存** ボタンをクリック
3. 保存ダイアログで `.object` ファイルの保存先を指定（初期ディレクトリ = 現在表示中のフォルダ）

## 状態保存先

`{AviUtl2 appdata}\Plugin\PreviewExplorer\state.ini`
