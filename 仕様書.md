# MediaExplorer 仕様書

## 概要

AviUtl2 の `.aux2` 汎用プラグイン。
プレビュー（左）とメディアエクスプローラー（右）を横並びにした複合ウィンドウを提供する。
スプリッタで左右の幅を可変にでき、状態は INI ファイルに記憶する。

---

## ウィンドウ構成

```
┌──────────────────────────────────────────────────────────┐
│ 左ペイン（プレビュー）  ║  右ペイン（メディアエクスプローラー）│
│                         ║                                  │
│  AviUtl2 プレビュー     ║  [大タブ A][大タブ B][大タブ C]  │
│  ウィンドウを           ║  [小タブ 1][小タブ 2][小タブ 3]  │
│  埋め込み（試験的）     ║  ┌──────────────────────────┐   │
│                         ║  │ クイックアクセス（折畳可）│   │
│                         ║  ├──────────────────────────┤   │
│                         ║  │ ファイルリスト（ListView）│   │
│                         ║  └──────────────────────────┘   │
└─────────────────────────╨──────────────────────────────────┘
                        ↑ 可動スプリッタ
```

---

## 機能仕様

### スプリッタ

- 左右ペインの境界に幅 5px の可動スプリッタを配置
- `WM_LBUTTONDOWN`〜`WM_LBUTTONUP` でドラッグ移動
- 左ペイン最小幅: 100px、右ペイン最小幅: 200px
- スプリッタ位置は INI ファイルに保存（`WM_LBUTTONUP` 時）

### 左ペイン（プレビュー）

- **Phase 1（実験）**: AviUtl2 プレビューウィンドウの HWND を探索して `SetParent` で埋め込み試行
  - `get_host_app_window()` でメインウィンドウを取得
  - `EnumChildWindows` で子ウィンドウを列挙しログ出力
  - 有望な候補に `SetParent` → リサイズして確認
- **失敗時**: 埋め込み失敗の診断情報をテキスト表示

### 右ペイン（メディアエクスプローラー）

#### タブ構造（2 層）

- **大カテゴリタブ**（上段）: ユーザー定義。例）動画 / 音声 / 画像 / その他
- **小カテゴリタブ**（下段）: 大カテゴリごとにカスタムフォルダセット
- タブ切り替え時に下のファイルリストを更新
- 選択中のタブ番号は INI ファイルに記憶

#### クイックアクセスパネル

- 大タブ・小タブの下に折りたたみ可能なパネル
- よく使うフォルダをボタン形式で表示
- トグルボタン（▲/▼）で表示/非表示を切り替え
- 表示状態は INI ファイルに記憶

#### ファイルリスト

- `ListView`（詳細表示）でファイルを表示
- ファイル名・種類・サイズ・更新日時 の列
- `is_support_media_file()` でフィルタリング（サポート外は薄く表示）
- ダブルクリックで `create_object_from_media_file()` を呼び出しタイムラインに挿入
  - 挿入位置: カーソル位置のレイヤー・フレーム
- フォルダクリックで下の階層へ移動、上段ボタンで親フォルダへ

## 状態保存（INI）

保存先: `{app_data_path}\Plugin\MediaExplorer\state.ini`

```ini
[Window]
SplitX=400
QuickAccessVisible=1

[Tabs]
BigTabIndex=0
SmallTabIndex_0=0
SmallTabIndex_1=0

[BigTab_0]
Name=動画
SmallTab_0_Name=BGM用
SmallTab_0_Path=E:\素材\BGM
SmallTab_1_Name=SE用
SmallTab_1_Path=E:\素材\SE

[RecentFolders]
Path_0=E:\素材\BGM
Path_1=E:\素材\動画
```

## ビルド環境

- Visual Studio 2022
- C++17
- CMake
- SDK: `src/sdk/` （`old_alias_save` プロジェクトから複製）

```powershell
cmake -B build -G "Visual Studio 17 2022" -A x64
cmake --build build --config Release
```

出力: `build/Release/MediaExplorer.aux2`
