# PreviewExplorer 仕様書

## 概要

AviUtl2 の `.aux2` 汎用プラグイン。
プレビュー（左ペイン）と Windows シェルエクスプローラー（右ペイン）を横並びにした複合ウィンドウを提供する。
スプリッタで左右の幅を可変にでき、全状態は INI ファイルに記憶する。

プラグイン名: **Preview edit + Explorer**
出力ファイル: `PreviewExplorer.aux2`

---

## ウィンドウ構成

```
┌──────────────────────────────────────────────────────────────────┐
│ 左ペイン（プレビュー）  ║  右ペイン（メディアエクスプローラー）    │
│                         ║  [大タブ A][大タブ B]...               │
│  AviUtl2 プレビュー     ║  [小タブ 1][小タブ 2]...               │
│  ウィンドウを           ║  ──────────────────────────────────    │
│  埋め込み（試験的）     ║  [←][→][↑][アドレスバー][コピー][切取][貼付][A保存] │
│                         ║  ┌──────────────────────────────────┐  │
│                         ║  │ IExplorerBrowser（シェルビュー）  │  │
│                         ║  └──────────────────────────────────┘  │
└─────────────────────────╨──────────────────────────────────────────┘
                        ↑ 可動スプリッタ（幅 5px）
```

## 機能仕様

### スプリッタ

- 左右ペインの境界に幅 5px の可動スプリッタを配置
- マウスドラッグで左右比率を変更
- 左ペイン最小幅: 100px、右ペイン最小幅: 200px
- スプリッタ位置比率（`SplitRatio`）を INI に保存（`WM_LBUTTONUP` 時）

### 左ペイン（プレビュー）

起動 2 秒後に AviUtl2 プレビューウィンドウを探索して `SetParent` で埋め込み試行する（試験的）。

- `get_host_app_window()` でメインウィンドウを取得
- 同プロセスのトップレベルウィンドウを列挙し、タイトルに「プレビュー」を含む HWND を候補とする
- 候補ウィンドウのスタイルを `WS_CHILD` に変更し、左ペインを親にして埋め込む
- 失敗した場合はウィンドウ一覧の診断テキストを表示

### 右ペイン（メディアエクスプローラー）

#### タブ構造（2 層）

- **大カテゴリタブ**（上段、ボールドフォント）: ユーザー定義のワークスペースグループ
- **小カテゴリタブ**（下段、通常フォント）: 大カテゴリごとのフォルダブックマーク
- タブごとに独立した「最後にいた場所（`currentPath`）」を保持
- タブ切り替え時は各タブの `currentPath` へ自動ナビゲート（なければ `path` 初期位置）
- タブはドラッグ＆ドロップで順序変更可能（大・小ともに対応）

#### タブ右クリックメニュー

**大タブ**:
| メニュー項目 | 動作 |
|---|---|
| 追加 | 新規大タブを末尾に追加（小タブ「クイックアクセス」付き） |
| 名前変更 | 入力ダイアログでタブ名を変更 |
| 初期位置に移動 | そのタブの `currentPath` へ移動（`currentPath` がなければ `path`） |
| 削除 | タブを削除（最後の 1 つは削除不可） |

**小タブ**:
| メニュー項目 | 動作 |
|---|---|
| 追加 | 新規小タブを末尾に追加 |
| 名前変更 | 入力ダイアログでタブ名を変更 |
| パス変更... | フォルダ選択ダイアログで初期パス（`path`）を変更 |
| 現在の場所をパスに設定 | 現在表示中のフォルダを `path` として保存 |
| 初期位置に移動 | そのタブの `currentPath` へ移動 |
| 削除 | タブを削除（最後の 1 つは削除不可） |

#### ナビゲーションバー

ナビバー高さ: 28px（固定）

| コントロール | 内容 |
|---|---|
| `←` | 戻る（`IShellBrowser::BrowseObject` + `SBSP_NAVIGATEBACK`） |
| `→` | 進む（`SBSP_NAVIGATEFORWARD`） |
| `↑` | 親フォルダへ移動 |
| アドレスバー | 現在フォルダを表示。Enter でナビゲート。Ctrl+C/X/V/Z/A 対応 |
| コピー | 選択ファイルをコピー（クリップボードに `DROPEFFECT_COPY`） |
| 切取 | 選択ファイルを切り取り（`DROPEFFECT_MOVE`） |
| 貼付 | クリップボードのファイルを現在フォルダへ貼り付け（`IFileOperation`。同フォルダの場合は自動リネーム） |
| A保存 | タイムライン選択オブジェクトのエイリアスを `.object` ファイルとして保存 |

#### IExplorerBrowser

- `CLSID_ExplorerBrowser` を使用してネイティブのシェルビューを埋め込む
- ナビゲーション完了時 (`IExplorerBrowserEvents::OnNavigationComplete`) にアドレスバーと `currentPath` を更新
- 仮想フォルダ（クイックアクセス等）は `SIGDN_NORMALDISPLAY` で表示名、`SIGDN_DESKTOPABSOLUTEPARSING` で保存パスを取得
- ナビゲーションボタン操作後は 300ms タイマーでフォールバック更新 (`TIMER_ADDR_UPDATE`)

#### キーボードフック

AviUtl2 本体の `TranslateAccelerator` が Ctrl+C/V 等を横取りする問題を回避するため、`WH_KEYBOARD_LL` でグローバルフックを設置。右ペインにフォーカスがある場合のみシェル操作に転送する。

#### エイリアス保存（A保存ボタン）

- `EDIT_HANDLE::call_edit_section_param` でタイムラインにアクセス
- `EDIT_SECTION::get_focus_object()` で選択オブジェクトを取得
- `EDIT_SECTION::get_object_alias()` でエイリアスデータ（UTF-8 テキスト）を取得
- 保存ダイアログの初期ディレクトリ = 現在表示中のフォルダ
- デフォルトファイル名 = プロジェクト名 + `.object`
- 拡張子フィルタ: `*.object`

## ダークモード

- `uxtheme.dll` の未公開 ordinal 135 (`SetPreferredAppMode(2)`) でプロセス全体を ForceDark に設定
- ordinal 133 (`AllowDarkModeForWindow`) でウィンドウごとにダークモードを有効化
- `DwmSetWindowAttribute(hwnd, DWMWA_USE_IMMERSIVE_DARK_MODE, ...)` でタイトルバーをダーク化
- タブ・ボタン・アドレスバーは `NM_CUSTOMDRAW` + テーマカラー (`g_clrBg`, `g_clrText`) で自前描画
- シェルビューには `SetWindowTheme(hwnd, L"DarkMode_Explorer", nullptr)` を適用

## 状態保存（INI）

保存先: `{app_data_path}\Plugin\PreviewExplorer\state.ini`

```ini
[Window]
SplitRatio=0.4000

[Tabs]
BigTabCount=2
BigTabIndex=0
SmallTabIndex_0=1
SmallTabIndex_1=0

[BigTab_0]
Name=動画
SmallTabCount=2
SmallTab_0_Name=クイックアクセス
SmallTab_0_Path=
SmallTab_0_CurrentPath=C:\Users\user\Videos
SmallTab_1_Name=素材
SmallTab_1_Path=E:\素材\動画
SmallTab_1_CurrentPath=E:\素材\動画\2024
```

| キー | 内容 |
|---|---|
| `SplitRatio` | 左ペイン幅の割合（0.05〜0.95） |
| `BigTabCount` | 大タブ数 |
| `BigTabIndex` | 選択中の大タブ番号 |
| `SmallTabIndex_N` | 各大タブの選択中小タブ番号 |
| `SmallTab_j_Name` | 小タブ表示名 |
| `SmallTab_j_Path` | 小タブ初期位置（ブックマーク） |
| `SmallTab_j_CurrentPath` | 小タブ最終表示パス（タブごとの位置状態） |

## ビルド環境

- Visual Studio 2022
- C++17
- CMake 3.10 以上

```powershell
cmake -B build -G "Visual Studio 17 2022" -A x64
cmake --build build --config Release
```

出力: `build/Release/PreviewExplorer.aux2`

リンクライブラリ: `user32 comctl32 comdlg32 shlwapi shell32 ole32 oleaut32 dwmapi`
